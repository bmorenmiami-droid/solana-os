```python
TEMPORAL_BRAINS = {
    "now": MiniMax,      # 2026 perspective

    "older": qwen,       # earlier perspective
}

def ask_with_consensus(question):
    responses = {}
    for label, brain in TEMPORAL_BRAINS.items():
        responses[label] = brain.think(question)
    
    agreements = find_agreements(responses)
    disagreements = find_disagreements(responses)
    
    return {
        "answer": consensus(responses),
        "confidence": len(agreements) / len(TEMPORAL_BRAINS),
        "disagreements": disagreements  # WHERE THE VALUE IS
    }
```

### The Key Insight

The DISAGREEMENT is where the value is:
- All agree → solid answer, high probability
- Brain 1 says X, others say Y → something changed between dates
- All disagree → the question needs examination
